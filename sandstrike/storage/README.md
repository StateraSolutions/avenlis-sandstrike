## 🔧 **Features**

### **Core Storage**
- **SQLite Database**: Fast, reliable local storage
- **Session Management**: Track red team testing sessions
- **Results Storage**: Store and analyze test results
- **Settings Persistence**: User preferences and configuration

### **Prompt Management**
- **Collections**: Organize prompts into logical groups
- **Categories**: Classify prompts by attack type
- **Severity Levels**: Rate prompts by potential impact
- **Import/Export**: Share prompt collections with your team
